```
  - Moderate padding (3.5 units)

  - 3 column layouts
```

## Interactive Element States

### Buttons

```
Primary (Watch Now):
  Rest:   Gold gradient + border
  Hover:  translateY(-4px) + glow
  Active: translateY(0px)

Secondary (Icons):
  Rest:   Glass card
  Hover:  translateY(-2px) + scale(1.05) + gold border
  Active: scale(0.95)

  Rest:   Text color
  Hover:  Gold color + underline
```

```
Episode Card:
  Rest:   Glass card
  Hover:  translateY(-4px) + gold border + play overlay
  Active: Navigate

  Rest:   Glass card
  Hover:  translateY(-8px) + glow + play button
  Active: Navigate
```

```

  Rest:   Muted color
  Hover:  Scale(1.1) + gold color

  Rest:   Secondary text
  Hover:  Primary text + glass background
```
